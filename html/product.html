What is SigmaCR

SigmaCR is a review aggregator. The following figure briefly describes what happens in SigmaCR.



Given a selected product SigmaCR is able to identify its features and score them according to customer reviews of that product all over the web. SigmaCR collects consumer reviews from various e commerce sites like Amazon, Walmart, Ebay and etc. These collected reviews are then processed using advanced information extraction techniques to identify what is the opinion of each customer about the features of that product. All the customer opinions are collected and Sigma CR will score the product features by aggregating these various opinions.

The following sample shows how SigmaCR has identified some features of a LED TV model. Given that the system has identified Durability, Picture Quality, Noise reduction, Ease of Use and etc. for features of a TV, it will show the user an aggregated score for each of those features computed by reading all the related reviews.
